Run these examples from the root directory of cgrparser.

Please note that most realistic C code samples would require running the C
preprocessor before passing the code to **cgrparser**; see the `README file
<https://github.com/eliben/cgrparser/blob/master/README.rst>`_ and
`this blog post
<https://eli.thegreenplace.net/2015/on-parsing-c-type-declarations-and-fake-headers>`_
more details.
